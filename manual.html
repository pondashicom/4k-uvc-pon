4K-UVC-PON 使い方

1本の4K UVCカメラ入力から複数のショットを切り出し、STANDBY / PROGRAM / CUT / AUTO / Tバーで操作する擬似マルチカメラスイッチャー試作版です。

1. このソフトでできること

4K-UVC-PON は、1台のUVCカメラ映像を元に複数の仮想ショットを作り、スイッチャー風の操作で切り替えるためのブラウザ試作版です。

  • UVCカメラ入力の表示
  • STANDBY / PROGRAM の2画面確認
  • Shot 1-4 の仮想ショット作成
  • ショット位置のドラッグ移動
  • CUT / AUTO / Tバーによる切替
  • PROGRAM専用の別ウィンドウ出力

2. 画面の見方

STANDBY

次に出したい映像を確認するための待機画面です。

PROGRAM

現在出力中の映像です。

Shot 1-4

4つの仮想ショットです。各ショットをクリックすると STANDBY に選ばれます。

FRAME

カメラ全景の上に、各ショットの切り出し枠が表示されます。枠をドラッグして位置を変えられます。

Tバー / CUT / AUTO

PROGRAM と STANDBY を切り替えるための操作エリアです。

3. 基本操作

STANDBYを選ぶ

以下の方法で STANDBY を選択できます。

  • 1 ～ 4 キー
  • 画面下の 1 ～ 4 ボタン
  • Shot 1-4 のプレビュー枠クリック
  • Shot 1-4 のタイトル帯クリック
数字キーやショットクリックで変わるのは STANDBYのみ です。PROGRAM はその場では切り替わりません。

CUT

即時に STANDBY と PROGRAM を入れ替えます。

  • Enter キー
  • CUTボタン

AUTO

現在の PROGRAM と STANDBY の間を自動でトランジションします。

  • Space キー
  • AUTOボタン

AUTO 完了後に STANDBY と PROGRAM が入れ替わります。

Tバー

Tバーをドラッグして手動でトランジションできます。端まで到達すると切替が確定し、STANDBY と PROGRAM が入れ替わります。

4. 色の意味

  • 緑 = STANDBY
  • 赤 = PROGRAM

5. PROGRAM別ウィンドウ

PROGRAM画面をダブルクリックすると、PROGRAM専用の別ウィンドウを開けます。

  • 別モニタ確認用の試作
  • 現在のPROGRAMを追従表示
  • もう一度ダブルクリックで閉じる

6. 想定している使い方

この試作は特に、4K入力からFHD相当のショットを複数切り出す使い方を想定しています。

  • 入力: 3840 x 2160
  • 各ショット: 1920 x 1080 相当で切り出し

これにより、1台のカメラでも擬似的に複数カメラ運用のような見せ方ができます。

7. 現在できないこと

  • カメラ選択UI
  • 設定保存 / 復元
  • AUTO秒数調整
  • 音声の取り扱い

8. 注意事項

  • カメラ使用許可が必要です
  • ブラウザによって挙動が異なる場合があります
